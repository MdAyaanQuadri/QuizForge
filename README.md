# QuizForge

QuizForge is a quiz platform where a creator can build quizzes, manage questions, share a joining code, and participants can join, attempt the quiz, submit answers, and view the leaderboard.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Zod

## Features

- User signup, login, logout, and profile
- Quiz create, update, delete, and fetch
- Question create, bulk create, update, delete, and fetch
- Creator-only protection for quiz and question management
- Quiz joining with unique participant names
- Participant quiz attempt flow
- Answer submission with score calculation
- Leaderboard ranking
- Expiry handling for quizzes

## API Modules

- Auth
- Quizzes
- Questions
- Participants

## Status

Backend is completed for the current project scope. Frontend is the next step.
